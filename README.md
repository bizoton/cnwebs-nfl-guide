# cnwebs-nfl-guide

该项目是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容以信息展示和导航为主，不针对任何特定域名或具体网站。

## 项目简介

cnwebs-nfl-guide 旨在收集和整理一系列独立的 HTML 页面，这些页面可能包含各类信息、工具或参考资料。仓库以静态 HTML 文件的形式进行存储和版本管理，方便用户直接浏览或通过 GitHub Pages 等托管服务进行部署。

## 目录说明

仓库结构如下：

```
├── README.md          # 本文件
├── index.html         # 主入口页面（如有）
├── pages/             # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 静态资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放所有独立的 HTML 页面，每个文件为一个独立单元。
- `assets/`：存放页面所需的公共资源文件，按类型分类。

## 页面归档说明

每个 HTML 页面均为独立文件，不依赖外部接口或特定服务器环境。页面内容可能包括：

- 信息整理与展示
- 简单工具或计算页面
- 导航或索引页面

归档页面遵循以下原则：

- 不包含恶意代码或追踪脚本
- 尽量保持轻量和兼容性
- 不绑定任何特定域名或链接

## 维护说明

本仓库由维护者不定期更新。如果您发现页面内容有误或希望贡献新的 HTML 页面，欢迎提交 Pull Request 或 Issue。

贡献时请注意：

- 新增页面请放置在 `pages/` 目录下
- 尽量保持文件命名简洁清晰
- 如有资源依赖，请放入 `assets/` 对应目录
- 避免引入外部不可控资源（如第三方 CDN）

## 使用方式

您可以直接通过浏览器打开仓库中的 HTML 文件进行查看，也可以将整个仓库部署到静态托管服务上使用。
